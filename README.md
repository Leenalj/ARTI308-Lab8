
KNN Project
Overview

This project implements the K-Nearest Neighbors (KNN) algorithm for classification using Python and Scikit-learn.

Dataset
File: KNN_Project_Data.csv
Contains numerical features and a target column (TARGET CLASS)
Steps
Load and explore data (EDA)
Standardize features using StandardScaler
Split data into training and testing sets
Train KNN model
Evaluate using confusion matrix and classification report
Optimize K using error rate plot
Results
K = 1 → Accuracy ≈ 72%
K = 30 → Accuracy ≈ 83%
Technologies

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Conclusion

Model performance improved after selecting the optimal K value. Feature scaling and tuning K are essential for KNN.
